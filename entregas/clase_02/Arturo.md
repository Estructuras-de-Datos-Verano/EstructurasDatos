    1. Las estructuras de datos nos ayudan a organizar, orientar y hacer mas eficientes las tareas deseadas; Para cada tarea odemos encontrar una estructura mas optima, por lo que no existe una estructura que sea "superior". 

    Por  ejemplo, para buscar pertenencia es mejor un set pues a nivel complejidad es mucho mas facil (constante de hecho), por lo que aunque tengamos muchisimos datos siempre tardara muy poco, pero si queremos mas datos, no solo pertenencia, por ejemplo frecuencia, no podemos acceder a ella; por lo que para usar una estructura necesitamos tener bien definida la tarea que queremos optimizar.
