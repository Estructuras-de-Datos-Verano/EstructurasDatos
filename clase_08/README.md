# Clase 08: Laboratorio de investigacion tecnica

Esta sesion es asincronica y de fin de semana. No es una clase tradicional con notebook, presentacion, codigo o pruebas.

El objetivo es ampliar el mapa tecnico del curso: revisar recursos externos, comparar explicaciones, identificar patrones de problemas y preparar mejores preguntas para las siguientes clases.

## Pregunta guia

Como aprende una persona a reconocer que estructura de datos o patron algoritmico necesita un problema?

## Objetivos

Al terminar esta actividad podras:

- Buscar y seleccionar recursos tecnicos con criterio.
- Comparar explicaciones de estructuras de datos y algoritmos.
- Distinguir ejemplos, patrones y recetas superficiales.
- Relacionar recursos externos con los problemas trabajados en clase.
- Construir un catalogo inicial de patrones utiles.
- Formular preguntas tecnicas para futuras sesiones.

## Que debes consultar

Debes consultar al menos 8 recursos:

- Al menos 4 videos.
- Al menos 2 recursos escritos.
- Al menos 2 recursos encontrados por ti fuera de la lista sugerida.
- Al menos 1 recurso en ingles.
- No todos pueden venir del mismo autor, canal o sitio.

La lista sugerida esta en:

```text
recursos_sugeridos.md
```

## Entrega

Trabaja en:

```text
entregas/
└── clase_08/
    └── nombre/
        ├── revision_bibliografica.md
        ├── patrones.md
        └── reflexion.md
```

Usa las plantillas ubicadas en:

```text
plantillas/
```

## Actividad GitHub

1. Crea una rama con el nombre `clase-08-tu-nombre`.
2. Crea tu carpeta dentro de `entregas/clase_08/nombre/`.
3. Copia las tres plantillas a tu carpeta de entrega.
4. Completa los documentos con tus respuestas.
5. Haz commits claros.
6. Abre un Pull Request.
7. Revisa el Pull Request asignado por `asignar_revisiones.py`.

En la revision comenta sobre claridad de escritura, calidad de fuentes, comparacion critica, preguntas tecnicas y patrones identificados.

## Importante

No se entrega codigo.

No se ejecuta `pytest`.

No se entrega notebook.

La evidencia principal es tu capacidad de leer, comparar, organizar y argumentar tecnicamente.
