Clase abstracta como contrato

from abc import ABC, abstractmethod

class PilaAbstracta(ABC):
    @abstractmethod
    def push(self, elemento):
        pass

Contrato

Una subclase concreta debe implementar todos los métodos abstractos.

PilaAbstracta no es una pila concreta

No se instancia

No queremos hacer PilaAbstracta().

Sí obliga

Define qué métodos debe tener toda pila concreta.

TypeError: Can't instantiate abstract class ...

Contrato completo

class PilaAbstracta(ABC):
    @abstractmethod
    def push(self, elemento): ...
    @abstractmethod
    def pop(self): ...
    @abstractmethod
    def peek(self): ...
    @abstractmethod
    def esta_vacia(self): ...
    @abstractmethod
    def tamano(self): ...

PilaLista

push(elemento):
    agregar elemento al final

pop():
    quitar y regresar último elemento

peek():
    regresar último sin quitarlo

Ventajas

Simple, natural en Python, eficiente al final.

Desventajas

Depende del arreglo dinámico interno de list.

PilaDeque

from collections import deque

push(elemento):
    agregar a la derecha

pop():
    quitar de la derecha

Ventajas

Diseñada para operaciones eficientes en extremos.

Desventajas

Oculta detalles internos y puede ser menos clara al inicio.

PilaTupla

push(elemento):
    crear nueva tupla con elemento al final

pop():
    guardar último
    reemplazar por tupla sin último

Ventajas

Permite hablar de inmutabilidad.

Desventajas

Cada cambio crea una nueva tupla.

Polimorfismo

La misma función puede usar objetos distintos si respetan la misma interfaz.

PilaLista PilaDeque PilaTupla misma función

Función que usa cualquier pila

def vaciar_pila(pila):
    elementos = []
    while not pila.esta_vacia():
        elementos.append(pila.pop())
    return elementos

La función no necesita saber si la pila usa list, deque o tuple.

Transferir elementos

def transferir_elementos(origen, destino):
    while not origen.esta_vacia():
        destino.push(origen.pop())

Cualquier pila.

Destino

import pytest

def test_pop_en_vacia_lanza_error():
    pila = PilaLista()
    with pytest.raises(IndexError):
        pila.pop()

pytest.mark.parametrize

@pytest.mark.parametrize("ClasePila", [
    PilaLista,
    PilaDeque,
    PilaTupla,
])
def test_pop_respeta_lifo(ClasePila):
    pila = ClasePila()
    pila.push(1)
    pila.push(2)
    assert pila.pop() == 2
