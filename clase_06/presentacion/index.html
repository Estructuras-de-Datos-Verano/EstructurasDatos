Problema central

Josephus Problem I

CSES Problem Set · cses.fi/problemset/task/2162

Hay `n` niños numerados en círculo. Se elimina uno de cada dos hasta que no queda nadie. Debemos imprimir el orden de eliminación.

¿Por qué Josephus?

Simulación

El proceso se puede ejecutar paso a paso.

Cola natural

FIFO aparece por operaciones, no por definición.

Complejidad

`n` llega a `2·10^5`, así que importa la eficiencia.

Bloque 1: leer sin programar

Entrada, salida, restricciones

Entrada

Un entero `n`.

Salida

`n` enteros: orden de eliminación.

Restricción

`1 <= n <= 2·10^5`.

Ingeniería inversa del algoritmo

  1. ¿Qué pide exactamente?
  2. ¿Qué información debo recordar?
  3. ¿Qué operaciones realizo continuamente?
  4. ¿Qué estructura implementa esas operaciones?
  5. ¿Cómo lo haría con papel y lápiz?
  6. Solo después: pseudocódigo.

1. ¿Qué está pidiendo?

No pide quién gana. Pide todo el orden de eliminación.

longitud de salida: n cada niño aparece una vez el orden importa

2. ¿Qué debo recordar?

Estado vivo

Niños que siguen participando y su orden circular.

Resultado

Lista de eliminados en el orden exacto.

3. Operaciones repetidas

  1. Tomar al siguiente niño.
  2. Si se salva, mandarlo al final.
  3. Tomar al siguiente niño.
  4. Eliminarlo.
  5. Guardar el eliminado.

Descubrimos la cola

El frente se procesa; quien se salva vuelve al final.

frente -> se salva -> final
frente -> se elimina -> respuesta

¿Por qué no una pila?

Pila

Sale lo último que entró. Rompe el orden circular natural.

Cola

Atiende el frente y permite reinsertar al final.

Algoritmo en español

Crear una cola con 1, 2, ..., n.
Mientras haya niños:
    mover al final al niño que se salva
    eliminar al siguiente niño
    guardar el eliminado
Regresar la lista de eliminados

Pseudocódigo

función orden_eliminacion(n):
    vivos = cola(1..n)
    eliminados = []

    mientras vivos no esté vacía:
        si solo queda uno:
            eliminarlo
        si no:
            mover frente al final
            eliminar nuevo frente

    regresar eliminados

Problema secundario

Nearest Smaller Values

CSES Problem Set · cses.fi/problemset/task/1645

Para cada posición, encontrar la posición más cercana a la izquierda con valor menor.

¿Por qué una cola deja de funcionar?

Josephus

El orden de atención es circular y FIFO.

Nearest Smaller
